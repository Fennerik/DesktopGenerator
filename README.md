# DesktopGenerator
A simple bash script that helps generating .desktop files

## Useage
1. Clone this repository or download source code archive and extract it
2. Run ```./desktop nameofyourdesktopfile```
3. Replace _nameofyourdesktopfile_ with your actual name
4. Enter the details for the desktop file in the editor
5. Press Ctrl+X, select "save" and exit

_Note: This script needs nano to be installed. Make sure it is installed_
